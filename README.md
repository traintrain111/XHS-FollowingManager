# 小红书关注整理助手 ✨

一个用于整理小红书已关注博主的 Chrome 扩展：批量搜集、自动分类、手动改标签、收藏、搜索和备注，也支持通过笔记评论区 `@` 候选名单补抓已关注用户。

数据只保存在浏览器本地，不上传服务器，也不接入外部模型。

---

## 安装 🚀

1. 下载并解压项目。
2. 打开 Chrome：`chrome://extensions/`。
3. 开启右上角 `开发者模式`。
4. 点击 `加载已解压的扩展程序`。
5. 选择项目里的 `dist/` 文件夹。
6. 打开或刷新小红书网页，右上角会出现插件面板。

更新插件后，记得在扩展页点一次 `重新加载`，再刷新小红书页面。

---

## 怎么用 🧭

```text
登录小红书网页版
→ 搜索一个高频词，例如 `的`、`小红书`
→ 勾选“已关注”
→ 点“自动滚动搜集”
→ 点“二次搜集”
→ 确认是否自动分类
→ 到“标签”或“收藏夹”里检查和微调
```

如果搜集不全，可以换几个关键词重复搜集，例如 `阅读`、`穿搭`、`摄影`。插件会自动去重。

或者选用 `@名单采集` 方法，更全面，但存在一定风控风险：

```text
回到小红书首页
→ 打开插件
→ 点“@名单采集”
→ 插件会自动打开第一篇笔记
→ 展开评论输入区并点 `@`
→ 读取候选名单
→ 逐个打开用户搜索页，只保留已关注且匹配的用户
```

这条链路适合补抓那些没有稳定出现在搜索结果页里的已关注账号。

## 主要功能 ✅

- `扫描当前页`：只扫描当前已加载结果，适合快速测试。
- `自动滚动搜集`：自动下滑页面并入库更多已关注博主。
- `@名单采集`：从首页第一篇笔记的评论区 `@` 候选名单出发，再跳到用户搜索页识别已关注用户，适合补漏。
- `二次搜集`：慢速打开未分类博主主页，尝试补抓简介并辅助分类。
- `标签`：创建、删除、查看分类，也能查看 `无标签` 博主。
- `收藏夹`：快速查看已收藏博主。
- `搜索`：支持按昵称、主页链接、标签、备注查找。
- `备注`：给博主补充自己的判断信息。
- `关注切换`：可在卡片上尝试关注 / 取消关注。

---

## 数据说明 💾

插件会在本地保存：

- 博主信息
- 标签
- 收藏状态
- 备注
- 抓到的头像和主页简介

退出小红书账号、刷新网页不会清空数据；但卸载扩展、清理扩展数据、换浏览器或换电脑后，数据可能丢失。

---

## 使用提醒 ⚠️

- 小红书是懒加载页面，插件不能保证一次抓全所有关注博主。
- `@名单采集` 依赖首页笔记弹窗、评论输入区、`@` 按钮和用户搜索页结构，页面改版后可能失效。
- 二次搜集会打开博主主页，有一定风控风险，不建议连续高频使用。
- 自动分类不是 100% 准，适合先粗分，再手动微调。
- 如果小红书页面改版，扫描、滚动、简介抓取或关注切换可能失效。
- 关注 / 取消关注是真实站内操作，请确认后再点。
